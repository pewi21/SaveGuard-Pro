# Privacy Policy – SaveGuard Pro

SaveGuard Pro does not collect, store, or transmit any personal data.

## Data Collection

This plugin does NOT:

* collect personal information
* track user activity
* use analytics
* send data to external servers

## Local File Access

SaveGuard Pro requires access to the local file system only to:

* save documents
* create backup files
* manage backup storage

All operations are performed locally on the user's machine.

## Third-Party Services

SaveGuard Pro does not use any third-party services, APIs, or cloud integrations.

## Data Security

All data remains under the full control of the user and is never transmitted outside of the local environment.

## Changes

If this policy changes in the future, updates will be reflected in this document.
